🟢 1. خوارزميات البحث (Searching Algorithms)

Linear Search (البحث الخطي): البحث عنصرًا عنصرًا.

Binary Search (البحث الثنائي): البحث في مصفوفة مرتبة عن طريق تقسيم النطاق إلى نصفين.

Hashing (التجزئة): البحث باستخدام هياكل بيانات مثل HashMap.

🟢 2. خوارزميات الترتيب (Sorting Algorithms)

Bubble Sort (الفقاعات).

Selection Sort (الاختيار).

Insertion Sort (الإدراج).

Merge Sort (الدمج).

Quick Sort (السريع).

Heap Sort (الكومة).

Counting / Radix / Bucket Sort (للبيانات العددية الكبيرة أو الموزعة).

🟢 3. خوارزميات البرمجة الديناميكية (Dynamic Programming)

Fibonacci Sequence (بناءً على الحلول الجزئية).

Knapsack Problem (حقيبة الظهر).

Longest Common Subsequence (LCS).

Matrix Chain Multiplication.

Coin Change Problem.

🟢 4. خوارزميات الجشع (Greedy Algorithms)

Huffman Coding (ضغط البيانات).

Activity Selection Problem.

Minimum Spanning Tree (Prim’s & Kruskal’s).

Dijkstra’s Algorithm (أقصر مسار).

🟢 5. خوارزميات الرسوم البيانية (Graph Algorithms)

DFS (Depth-First Search).

BFS (Breadth-First Search).

Dijkstra’s Algorithm.

Bellman-Ford Algorithm.

Floyd-Warshall Algorithm.

Kruskal & Prim (MST).

Topological Sort.

🟢 6. خوارزميات الهياكل البيانية (Data Structures)

Stack & Queue Algorithms.

Binary Search Tree Traversals (Inorder, Preorder, Postorder).

Heap Operations.

Union-Find (Disjoint Set).

Segment Tree / Fenwick Tree (للعمليات على المصفوفات الكبيرة).

🟢 7. خوارزميات البحث في الحل (Search / Decision)

Backtracking (مثل حل Sudoku أو N-Queens).

Branch & Bound.

Binary Search on Answer (Optimization Problems).

🟢 8. خوارزميات التحسين (Optimization)

Dynamic Programming Optimization (Knuth, Divide & Conquer).

Linear Programming (خوارزمية Simplex).

🟢 9. خوارزميات الوقت والتاريخ

🟢 10. خوارزميات الأعداد الكبيرة (Big Integer Algorithms)

Fast Exponentiation (Exponentiation by Squaring).

🟢 11. خوارزميات المحاكاة (Simulation)

محاكاة الطوابير، الأحداث، أو حركة الكائنات.

🟢 12. خوارزميات الاحتمالات والإحصاء

Randomized Algorithms.

Sampling.

Markov Chains.

🟢 13. خوارزميات التشفير (Cryptography)

RSA Algorithm.
